Comparing on hover background color change

Transition

.transition-example {
	transition: 2s background;
}

.transition-example:hover {
	background: red;
}

Key Frames

.key-frame-example {
	animation-name: background-to-blue;
	animation-duration: 2s;
	animation-fill-mode: both;
}

.key-frame-example:hover {
	animation-name: background-to-red;
}

@keyframes background-to-red {
	100% {background: red;}
}

@keyframes background-to-blue {
	0% {background: red;}
	100% {background: blue;}
}
