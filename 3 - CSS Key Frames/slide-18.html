CSS Key Frames

Set what happens to the styles before
and after the animation

.none
.forwards
.backwards
.both
@keyframes animate {
	0% {
		background: green;
	}
	100% {
		background: blue;
	}
}

.none {
	animation-fill-mode: none;
}

.forwards {
	animation-fill-mode: forwards;
}

.backwards {
	animation-fill-mode: backwards;
}

.both {
	animation-fill-mode: both;
}
