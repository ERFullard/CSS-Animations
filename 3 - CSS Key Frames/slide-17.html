CSS Key Frames

Utilizing different properties and steps

@keyframes animate {
	0% {
		background: blue;
	}
	25% {
		transform: rotate(45deg);
		border-radius: 0%
	}
	50% {
		transform: rotate(45deg);
		border-radius: 50%;
		background: blue;
	}
	75% {
		background: red;
		transform: scale(1);
	}
	100% {
		transform: scale(2);
		border-radius: 50%;
	}
}
